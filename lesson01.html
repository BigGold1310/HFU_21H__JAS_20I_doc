Einführung, IDE Aufsetzen & Grundlagen

JavaScript 1.0

  • Als "LiveScript" in 1995 von Brendan Eich in 10 Tagen für Netscape entwickelt
  • Ursprünglich benutzt, um Java Applets in den Netscape Navigator einzubinden
  • Deshalb in JavaScript umbenannt

JavaScript 2.0

  • 1996 integrierte Internet Explorer 3.0 eine eigene Version namens JScript
  • Zeit des ersten Browserkriegs
  • Vernachlässigung von Standards, viele Inkompatibilitäten

Standardisierung

  • 1997 als ECMA-262 standardisiert
  • Browserseitige Schnittstellen vom W3C (World Wide Web Consortium) standardisiert.
  • Seit 2015 gab es pro Jahr eine neue Version
  • "Evergreen" Browsers werden automatisch aktualisiert (z.B. Chrome, Edge, Opera, Firefox aber nicht Safari)

Vorteile

  • Logik im Client spart Roundtrips und Rechenzeit auf dem Server
  • Schnellere Antwortzeiten für den Client

Nachteile

  • Browserinkompatibilitäten (immer weniger)
  • JavaScript kann deaktiviert werden (und nicht alle Browser unterstützen JavaScript)
  • Sicherheitsprobleme (fremder Code wird auf dem eigenen Computer ausgeführt, Ausbruch aus der Sandbox)

Erster Blick

  • Im Browser Console
  • Aus einer HTML-Datei
  • Aus einer JS-Datei
  • Im System Console (Node)
  • Testing (Jest)

Deklarieren: var

          
function varTest() {
  var x = 1;
  if (true) {
    var x = 2;  // gleiche Variable!
    console.log(x);  // 2
  }
  console.log(x);  // 2
}

Deklarieren: let

          
function letTest() {
  let x = 1;
  if (true) {
    let x = 2;  // andere Variable
    console.log(x);  // 2
  }
  console.log(x);  // 1
}

Deklarieren: const

          
function constTest() {
  const x = 1;
  if (true) {
    const x = 2;  // andere Variable
    console.log(x);  // 2
  }
  x = 2;  // Nicht erlaubt!
  console.log(x);  // 1
}

Zeichenketten (Strings)

Multi-line Strings

Für Multi-line Strings mit " und ' muss jede Zeile mit \ beenden. Für die mit Backticks nicht.

          
const a = "Das \
  ist \
  multi-line.";

const a = `Das
  ist
  multi-line.`;

Strings mit Variablen

Backticks unterstützen das Auflösen von eingebauten Variablen ("Interpolation").

          
const name = "Bob";
const a = "Das \
  ist " + name + " \
  multi-line.";

const a = `Das
  ist ${name}
  multi-line.`

Arrays

  • Ein Liste von Werten mit irgendwelchen Typen (Zahlen, Strings, Objekte, andere Arrays, usw.).
  • Kann "Sparse" oder "Dense" sein.
  • Kann mit initialen Werten gefüllt werden.
          
            const empty1 = [];

            const filled = [1, 2, 3];
            const sparse1 = [,,];         // sparse1.length == 3
            const sparse2 = new Array(3); // sparse2.length == 3
            const a = filled[1];          // a == 2

Operatoren

  • Rechenoperatoren
  • Vergleichsoperatoren
  • Boolesche Operatoren
  • Zuweisungsoperatoren
  • Spread (...) Operator

Rechenoperatoren

  • +, -, *, /, %, **
  • % berechnet den Restwert einer ganzzahligen Division (Modulo) 5 % 2 == 1.
  • 5 ** 2 is 5 hoch 2 = 25
  • ++ ist wie +1; -- ist wie -1
  • ++zahl erhöht zahl vor der Auswertung um 1.
  • zahl++ erhöht zahl nach der Auswertung um 1.
  • Negative Zahlen werden mit - geschrieben -zahl.
  • Es gilt Punkt vor Strich und dann links vor rechts.
  • Um die Reihenfolge zu ändern können Klammern () gesetzt werden.

Vergleichsoperatoren

  • ==, !=, >, <, <=, >=, ===, !==
          
            const x = 1 == '1';   // x ist 'true'
            const y = 1 === '1';  // y ist 'false'

Boolesche Operatoren

  • &&, ||, !
          
            const x = false;
            const y = true;
            const a = x && y; // a ist 'false'
            const o = x || y; // o ist 'true'
            const n = !x;     // n ist 'true'

Nullesche Operatoren

  • ?., ??
          
            const person = {
              first: "bob",
              last: "Hoffman",
              age: 34,
              company: {
                name: "Apple"
              }
            };
            const x = person?.middle; // x == undefined ohne Fehlermeldung
            const y = x ?? "nothing"; // y == "nothing"
            const z = person.company.name ?? "none"; // z == "Apple"
            const z2 = person?.company?.name ?? "none"; // z2 == "Apple"

Zuweisungsoperatoren

  • x = 1
  • x++, ++x
  • x--, --x
  • +=, -=, *=, /=, %=, &&=, ||=, ??=, **=
          
            let x = 1;
            const y = x++;  // y == 1
            const z = ++x;  // z == 3
            x = x + 1;      // x == 4
            x += 1;         // x == 5; Kurzschreibweise für x = x + 1;

Spread (...) Operator

Kopiert alle eigenschaften eines Objekts.

          
            const person = {
              a: 1,
              b: "n",
              c: false,
              d: {
                e: 2,
                f: "m"
              }
            };
            const person2 = person;                         // Referenz
            const personCopy1 = Object.assign({}, person);  // Kopie
            const personCopy2 = { ...person };              // Kopie
            person.a = 2;  // Ändert nur person und person2

Spread mit Anpassungen

Kann mit explizitem Setzen von Eigenschaften kombiniert werden.

          
            const person = { a: 1, b: "n", c: false, d: { e: 2, f: "m" } };
            const person2 = {
              ...person,
              a: 2,
              d: {
                ...person.d,
                e: 1
              }
            } // { a: 2, b: "n", c: false, d:  { e: 1, f: "m" } }

Spread mit Arrays

Kopiert nicht nur Objekte sondern auch Arrays.

          
            const a = [0, 1, 2];
            const b = [3, 4, 5];

            const b2 = [...b];            // Kopie von b

            const c = [...a, ...b];       // [0, 1, 2, 3, 4, 5, 6]
            const d = [0, 1, 2, ...b, 7]; // [0, 1, 2, 3, 4, 5, 6, 7]

Spread mit Parametern

Kann auch als Funktionsparametern benutzt werden.

          
            function sum(x, y, z) {
              return x + y + z;
            }
            const numbers = [1, 2, 3];
            const total = sum(...numbers);  // Ergibt 6

Statements

  • Schleifen: for

Verzweigungen: if/else

          
            const x = 42;
            let y;
            if (x > 42) {
              y = 1;
            } else {
              y = 2;
            }
            // y == 2

Verzweigungen: switch

          
            const x = 42;
            let y;
            switch (x) {
              case 41:
                y = 1;
              case 42:
                y = 2;
              default:
                y = 3;
            }
            // y == 2

Schleifen: for

Für sich wiederholende Programmteile werden Schleifen verwendet.

          
            for (let i = 0; i < 5; i++) {
              // Wird für i = 0,1,2,3,4 ausgeführt
            }

Schleifen: for..in

Um allen Eigenschaften eines Objekts durchzugehen.

          
            const o = { a: 1, b: 2, c: 3 };

            for (const property in o) {
              // Ergibt "a", "b", "c"
            }

Schleifen: for..of

Um allen Elementen eines Arrays durchzugehen.

          
            const a = ['a', 'b', 'c'];

            for (const element of array1) {
              // Ergibt 'a', 'b', 'c'
            }

Schleifen: while

Um Code auszuführen bis eine Bedingung erfüllt wird.

          
            let n = 0;

            while (n < 3) {
              n++;
            }
            // n ist 3

Schleifen: do..while

Um Code mindestens ein mal auszuführen und dann bis eine Bedingung erfüllt wird.

          
            let result = '';
            let i = 0;

            do {
              i = i + 1;
              result = result + i;
            } while (i < 5);
            // result ist "12345"

Gewöhnliche Funktionen

  • Kann mehrere Parameter haben
  • "rest" Parameter wird mit "..." geschrieben
  • Parametern können default-Werte haben
          
            function add(a, b, c = 0) {
              return a + b + c;
            }
            function addAll(...args) {
              let result = 0;
              for (const a of args) {
                result += a;
              }
              return result;
            }
            const sum1 = add(1, 2);           // 3
            const sum2 = add(1, 2, 3);        // 6
            const sum3 = add(1, 2, 3, 4);     // 6 weil "4" ignoriert wird
            const sum4 = addAll(1, 2, 3, 4);  // 10
            const sum5 = addAll(...[1,2,3]);  // 6

Methoden

Funktionen, die an einem Objekt gebunden sind.

          
            const person = {
              age: 34,

              setAge(a) {
                this.age = a;
              }

              getYearsLeft() {
                return 65 - this.age;
              }
            };

            person.setAge(42);
            const y = person.getYearsLeft();  // 23

Getters/Setters

Ermöglich berechnete Eigenschaften.

          
            const person = {
              _age: 34;

              get age() {
                return _age;
              }

              set age(a) {
                this._age = a - 10; // Don't lie about your age 😉...
              }
            };

            person.age = 42;
            const y = person.age;  // 32

Arrow Funktionen

  • Eine kompaktere Definitionsform für Funktionen.
  • Für Methoden nicht gut geignet (weil this eine andere Bedeutung hat).
          
            const add = (a, b, c = 0) => a + b + c;
            const addAll = (...args) => {
              let result = 0;
              for (const a of args) {
                result += a;
              }
              return result;
            }
            const singleArgument = a => a + 2;  // Benötigt keine Klammern
            const noParameters = () => false;   // Benötigt leere Klammern

Umwandlungsfunktionen

  • parseInt()
  • parseFloat()
          
            const x = "42.3";
            const xType = typeof x;   // "string"
            const y = parseFloat(x);  // y == 42.3
            const yType = typeof y;   // "number"
            const z = parseInt(x);    // z == 42
            const zType = typeof z;   // "number"

Array-Methoden (mutating)

Die folgende Methoden ändern das Array ("mutating methods").

  • push() Element am Ende hinzufügen
  • pop() Element am Ende entfernen
  • unshift() Element am Anfang hinzufügen
  • shift() Element am Anfang entfernen
  • splice() Element im Array hinzufügen oder löschen
  • sort() Elemente sortieren
  • reverse() Reihenfolge der Elemente umkehren

Array-Methoden (immutable)

Die folgende Methoden ändern das Array nicht ("immutable methods").

  • slice() Erzeugt ein neues Array aus bestimmten Elementen des ersten
  • concat() Arrays verbinden
  • join() Erzeugt einen String aus allen Elementen
  • includes() Prüft, ob ein Array ein bestimmtes Element enthält
  • indexOf() Gibt den Index zurück, an dem ein bestimmtes Element im Array

Array-Methoden (immutable)

Die folgende Methoden ändern das Array nicht ("immutable methods").

  • find() Gibt den Wert, welches als erstes die Bedingung einer Funktion erfüllt
  • map() Wendet auf jedes Element des Arrays eine Funktion an und gibt das Ergebnis in einem neuen Array zurück.
  • reduce() Ähnlich zu map(), aber mit Akkumulator
  • filter() Erstellt ein neues Array mit allen Elementen, die den von einer Funktion implementierten Bedingung bestehen.
